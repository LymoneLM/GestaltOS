# 格式塔系统 GestaltOS

## 关于

《格式塔系统》 (GestaltOS) 是一款融合了 放置、战略、MMO 等元素的轻量**自动化**游戏。

更多介绍详见我们的 [设计文档](./docs/general.md)

本项目远程仓库地址 [GitHub](https://github.com/LymoneLM/GestaltOS)

本项目依据下文开源许可说明开源，将长期作为开源游戏存在。

## 如何开始

现在仍然是一个空壳项目，所以……

1. 克隆本仓库
2. 任选一个您喜欢的编程语言
3. 配置运行环境
4. 写个Hello World
5. 运行吧
